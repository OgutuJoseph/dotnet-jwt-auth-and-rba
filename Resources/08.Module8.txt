Module 8 - Evaluating user claims
• To generate a 'sample' token with player role-claims for a specific player as well as subscription claim
    [Inside dotnet-jwt-auth-and-rba >> GamesApi]
    -> dotnet user-jwts create --role "player" -n player1 --claim "subscription=gold"

• To view token details
    [Inside dotnet-jwt-auth-and-rba >> GamesApi]
    -> dotnet user-jwts print [tokenId :: e.g 3885c0bf]
